Interfaz de servicios web¶

El servicio iCMS proporciona una interfaz a través del protocolo HTTP que permite interactuar con las colecciones, posibilitando operaciones de descubrimiento, búsqueda, sincronización y suscripción a las colecciones que gestiona.

Es posible la comunicación con la interfaz mediante una comunicación de tipo REST o SOAP. Para más información, véanse los apartados Interfaz HTTP e Interfaz SOAP.
