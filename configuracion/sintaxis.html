Sintaxis¶

iCMS utiliza para interpretar los ficheros de configuración los métodos proporcionados por la clase ConfigSlurper, que permiten una sintaxis de configuración más potente que la de un fichero properties convencional.

Las ventajas que tiene utilizar este tipo de configuración son varias:

  • Estructura en árbol de las opciones de configuración, que aportan más claridad y organización.
  • Posibilidad de tipar las opciones de configuración.
  • La posibilidad de definir variables asociadas a un entorno determinado.

Advertencia

Es necesario recordar definir los valores de tipo String utilizando comillas simples o dobles.
