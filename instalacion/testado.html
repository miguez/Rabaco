Batería de pruebas¶

El proyecto icms-server cuenta con una amplia batería de pruebas que comprueba y garantiza la funcionalidad del sistema. Estos tests están basados en la conocida librería de testeo JUnit, por lo que pueden ser lanzados utilizando la herramienta de línea de comandos Apache Maven. Para ello, basta con escribir desde el directorio raíz del proyecto icms-server el siguiente comando

mvn test

Es posible de igual manera la ejecución de la batería de tests desde un entorno de desarrollo como Eclipse o Netbeans.
